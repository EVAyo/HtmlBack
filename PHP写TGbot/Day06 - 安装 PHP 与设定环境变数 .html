【PHP Telegram Bot】Day06 - 安装 PHP 与设定环境变数

用 PHP 打造专属於自己的 Telegram 聊天机器人吧！ 系列 第 6 篇
miku3920
1 个月前 ‧ 171 瀏览
0

今天要来安装执行 PHP 程式的程式，PHP 的解释器（直译器）。

通常 PHP 都是跑在网站伺服器上，应该很少人知道 PHP 可以用终端机(Terminal)或是称作命令行模式(CLI)执行吧。

这个程式跟一般的程式的安装方法不太一样

就从最简单的 Linux 讲到最复杂的 Windows 吧

Ubuntu or Linux

安装 php7.4

sudo apt install php7.4

安装扩充套件

sudo apt install php7.4-cli php7.4-fpm php7.4-gd php7.4-json php7.4-xml php7.4-zip php7.4-mbstring php7.4-curl php7.4-openssl php7.4-mysql php7.4-pdo_mysql

查看 php 版本

php -v

如果有出现版本 PHP 7.4.23 (cli) 就表示安装成功

MacOS

在「工具程式」里找到「终端机(Terminal)」並且打开（就是那个人见人怕的小黑框）

  1. 安装 Homebrew

    首先要安装 MacOS 的套件管理程式 Homebrew

    在终端机贴上这段指令並按下 Enter 就会开始安装（可能会要你输入密码）

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

    如果出现 Press RETURN to continue or any other key to abort，就按 Enter 或任意键它就会继续安装了

    装完会出现 Installation successful!，可能还会有 Next steps，此时就要再输入它给的指令：

    echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"

    装完后可以输入 brew 看看

    brew

    如果出现一堆 brew 啥的，就表示安装成功囉

    如果是出现 command not found: brew，代表没装成功，请重来一遍~

  2. 安装 php@7.4

    用刚刚安装的 Homebrew 装 php

    brew install php@7.4

    设定环境变数（让你可以在终端机用 php 指令）

    export PATH=/usr/local/opt/php@7.4/bin:$PATH

    重新载入环境变数

    exec $SHELL

    最后输入 php -v

    php -v

    如果有出现版本 PHP 7.4.23 (cli) 就表示安装成功

    如果是出现 command not found: php，代表没装成功，请重来一遍~

Windows

  1. 下载 PHP

    安装档在这个位置，不是很好找：

    PHP 官网 > Downloads > Windows downloads > PHP 7.4

    推荐是装 Thread Safe 的版本，因为 Apache 上必须使用线程安全的 PHP，虽然现在没有要装 Apache，但还是免得以后麻烦要重新装过

    要找 zip 档案下载，其他的都不是。

    我已经帮大家都找好了，赶时间的话，直接点下方连结下载就行了：

    • php-7.4.23 x64 Thread Safe
    • php-7.4.23 x86 Thread Safe

    至於 x86 还是 x64 可以在 设定 > 系统 > 关於 > 系统类型 看到：

    在 C 槽建立一个 php-7.4.23 的资料夹（你也可以放在其他地方）

    把刚刚载下来的压缩档丟进去

    右键解压缩到当前资料夹

    这样就算是安装完了

  2. 设定环境变数

    设定环境变数后才能在终端机用 php 指令

    找到 设定 > 系统 > 关於 的右边 相关设定 > 系统保护，然后点下去

    点 进阶，之后点 环境变数

    找到 path 后，点一下，再按 编辑

    按 新增，把刚刚解压缩的资料夹路径贴进去按 Enter

    最后就按 確定 確定 確定 就行了

  3. 开启小黑框

    在下方搜寻框输入 cmd，之后开启 命令提示字元（终端机(Terminal)）

    输入 php -v

    php -v

    出现版本 PHP 7.4.23 (cli)，就表示设定成功了~

    如果出现 'php' 不是內部或外部命令、可执行的程式或批次档。，表示你的「环境变数」没设定成功，请再加油~
