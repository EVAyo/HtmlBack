【PHP Telegram Bot】Day11 - Webhook 与 Web Hosting

用 PHP 打造专属於自己的 Telegram 聊天机器人吧！ 系列 第 11 篇
miku3920
24 天前 ‧ 61 瀏览
0

网站伺服器

网路上有很多的免费伺服器(Free Hosting)

几乎每个都有支援 PHP

我就拿 000webhost 来示范一下好了

网站上方有个登入

我是直接按用 google 登入，你也可以选下方的 Sign Up 註册

选底下的跳过

这边会问你网站名称，还有设定一个密码

设定的名称会当作你的域名，我设定 miku3920，网站就会是 https://miku3920.000webhostapp.com

这边三个可以隨便挑，我喜欢自己刻所以选第三个

之后就会进到档案管理，public_html 就是网站的位置，点两下点进去

新增一个 php 档案，档名隨意，越不容易被猜到越好（因为任何人都能访问这个程式）

先把昨天写的东西註解掉（Long Polling 的部分）

还有把底下官方写的东西取消註解，因为现在是要用 Webhook 让程式运作嘛

最后把程式全部贴到刚刚的那个档案里，储存並关闭，这样就完成了

附注：档案管理在这里 Tools > File Manager

设定 Webhook

找到你网站的域名，这里可以看到

https://miku3920.000webhostapp.com

把刚刚档案的档名接在后面

https://miku3920.000webhostapp.com/AAEdySrbzTc8tVdBjfymlfyaBUJGeu7r-v4.php

填进 WEBHOOK_URL 的值

然后执行程式

Request was successful: Webhook was set

看到这个表示完成设定了

试著跟机器人说 Hi、Hello 吧

如果机器人有回你，就表示成功啦

取消 Webhook

要取消也很容易，只要將网址刪除

最后再执行一次程式就行了

Request was successful: Webhook was deleted

这样就取消 Webhook 了

结尾

伺服器还是推荐用付费的啦

不然就会像我这样没事就被停掉QQ

哭阿，最好是钓鱼网站啦

很久以前有用过满长一段时间的说

后来改用付费的就没再用过免费的了

另外再推荐一个台湾人自己的服务 https://lionfree.net/

这个是很久很久以前，我刚写程式时用的伺服器

虽然较常停机，但没有语言不通的问题，也不会没事把你的服务停掉

还有粉丝专页会发布各种讯息

感谢 Lionfree，讚嘆 Lionfree

到此就算是讲完架设机器人的部分啦\洒花/
