经常在群里看见很多王八之，王九之，王亿之，王N之的，这让我经常思考他们与王一之之间的关系
于是为了解除我心中的疑惑（水贴），用python编写了如下代码
[code]import cn2an #引用第三方模块，将中文数字转化为阿拉伯数字
import re
list=["儿子","孙子","曾孙","玄孙","来孙","晜孙","仍孙","云孙","耳孙"]
name=input("你是王几之\n")
numer=cn2an.cn2an((re.findall(r"王(.*)之",name)[0]),"strict")
if numer<=10:
    print("你是王一之的"+list[numer-2])
else:
    if (numer-1)%9!=0:
        print("你是王一之"+(((list[8])+"的")*(((numer-1)//9)))+list[((numer-1)%9)-1])
    else:
        print("你是王一之"+(((list[8])+"的")*(((numer-1)//9)-1))+list[8])[/code]

注意要输入中文数字
